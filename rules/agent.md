# Agent Rules

## Purpose
This agent assists in developing the Department Event Management System using MERN Stack and Supabase.

## General Rules
- Follow modular and scalable architecture
- Do not hardcode credentials or secrets
- Prefer reusable components and clean code
- Follow role-based access control strictly
- Ensure consistency across frontend, backend, and database

## Coding Standards
- Use meaningful variable and function names
- Keep files small and focused
- Add comments only where logic is complex
- Handle errors gracefully

## Forbidden
- No direct database access from frontend
- No skipping authentication checks
- No UI logic in backend
